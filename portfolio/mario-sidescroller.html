De opdracht

De opdracht was om een interactief spel te maken door gebruik te maken van HTML, CSS en JavaScript. Dit is de eerste keer in de opleiding dat we gebruik maken van JavaScript. Voordat we zijn begonnen moesten we een idee of spel uitkiezen die we (na)wilden maken en ik heb gekozen voor een sidescroller (Mario Bros). In het spel is het de bedoeling om van punt A naar punt B te komen met allemaal obstakels die je moet doorstaan, waaronder vijanden en valkuilen.

De uitvoering

Ik heb in de HTML een canvas neergezet en met CSS deze in het midden gezet en een achtergrond gegeven. Alle overige code is JavaScript. Alle assets zijn png's die ik gemaakt heb met Adobe XD. Deze zijn allemaal gebaseerd op de sprites uit Super Mario Bros. Voor alle assets heb ik een JavaScript in een variabele aangemaakt. Elk variabel zit een class. In de class zitten gegevens van de assets en deze worden opgeroepen door de draw() functie of de update() functie. De update() functie is voor variabelen die blijven bewegen op het scherm (bv. de vijanden). Deze functies worden opgeroepen met de animate() functie onderin de code. Eigen reflectie: Ik ben trots op wat ik heb gemaakt in zo'n korte tijd. Ik had hiervoor nog nooit JavaScript gebruikt, maar had wel snel de smaak te pakken. Achteraf had ik zeker nog nieuwe dingen toe kunnen voegen en bepaalde bugs eruit kunnen halen. De goombas worden niet beinvloed door zwaartekracht omdat ze anders door de platforms heen zouden vallen. Er is geen checkpoint systeem. Het 2e level heeft niet echt een einde. Dit zijn dingen die achteraf wel op te lossen waren.
