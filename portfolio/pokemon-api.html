De opdracht

Voor het vak API van de minor Web Design & Development was de opdracht om een website te maken die gebruik maakt van een API. Ik heb een Pokémon website gemaakt met behulp van PokéAPI. Je moest minimaal een overzichtspagina en een detailpagina hebben en gebruik maken van minstens 2 web API's.

De uitvoering

Naast een overzichtspagina en een detailpagina van alle Pokémon in generatie 1 in de stijl van Pokémon kaarten heb ik ook met een canvas een wereld waarin je rond kunt lopen en Pokémon kunt vangen. Deze pokemon worden lokaal opgeslagen en kun je zien tijdens de encounter.
